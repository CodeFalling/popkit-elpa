# popkit-elpa
popkit elpa是melpa的国内镜像，满足国内emacs用户快速安装包的需求:
项目首页为 http://elpa.popkit.org/

## INTRO
这个项目主要从[melpa](https://github.com/aborn/melpa)项目fork而来
## 引入
```elisp
(add-to-list 'package-archives
            '("popkit" . "http://elpa.popkit.org/packages/"))
```
